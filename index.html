1. Introdução

O projeto OpenPDF foi desenvolvido com o objetivo de facilitar o trabalho de profissionais que realizam digitalizações e uploads de documentos em larga escala diariamente. O sistema monitora uma pasta local e envia automaticamente arquivos adicionados para um sistema em nuvem, de forma segura e eficiente.

2. Documentação do Usuário

2.1 Objetivo

Automatizar o processo de backup de documentos escaneados para a nuvem.

2.2 Funcionalidades

  • Monitoramento de pasta local
  • Upload automático
  • Integração com scanner
  • Execução em segundo plano
  • Encerramento fácil

2.3 Benefícios

  • Redução de tempo
  • Facilidade de uso
  • Organização e segurança
  • Controle pelo usuário

3. Documentação Técnica

3.1 Estrutura do Projeto


├── desktop_app.py       # Inicializador principal
├── auth.py              # Autenticação
├── config.py            # Configurações
├── lixeira.py           # Organização
├── script.py            # Utilitários
├── templates/           # Interface
└── static/              # Recursos estáticos

3.2 Orientações para Desenvolvedores

  • UI em templates/
  • Lógica de monitoramento em desktop_app.py
  • Tokens em auth.py
  • Diretórios em config.py

5. Instalação

5.1 Requisitos

Python 3.8+

5.2 Passos


git clone https://github.com/OpenPDF0/OpenPDF.github.io.git
cd OpenPDF.github.io
python -m venv venv
source venv/bin/activate  # ou venv\\Scripts\\activate
pip install -r requirements.txt
python desktop_app.py
